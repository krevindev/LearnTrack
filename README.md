# LearnTrack

LearnTrack is a web app for tracking learning progress.
It helps users organize what they are learning and see their progress over time.

## What it does

- Track learning goals and skills
- Organize topics under each skill
- Monitor progress
- Works on desktop and mobile

## Built with

- React
- TypeScript
- Material UI
- Tailwind CSS
- Vite
- Firebase

## Getting started

Clone the repo

git clone https://github.com/krevindev/learntrack.git
cd learntrack

Install dependencies

npm install

Run the app

npm run dev

Open http://localhost:5173 in your browser
